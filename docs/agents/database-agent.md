# Database Agent

## Foco
- modelagem
- migrations
- naming de tabelas
- indices
- relacionamentos
- Prisma 7 com PostgreSQL 17

## Regras
- tabelas em `snake_case`
- nomear FKs de forma consistente
- criar indices para consultas operacionais reais
- preferir integridade por constraint quando possivel
- migrations devem ser pequenas, reversiveis e rastreaveis
