# Security Agent

## Foco
- validacao de entrada
- sanitizacao
- CORS
- rate limit
- auth futura
- secrets e env
- logs seguros

## Regras
- bloquear entrada invalida antes da camada de servico
- nunca registrar segredos ou dados sensiveis
- revisar privilegios de banco e acesso externo
- projetar auth futura sem acoplamento prematuro
